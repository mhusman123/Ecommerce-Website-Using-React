## Run Locally

Clone the project

```bash
  git clone https://github.com/mhusman123/Ecommerce-Website-Using-React.git
```

Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
